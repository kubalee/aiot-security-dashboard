# 安芯智慧安防 AIoT 联动中枢

Vue 3 + Vite 高保真产品原型，用于展示安防 AI 设备上报、平台 AI 研判、物联系统联动、人员通知、PC/手机看板与整体统计分析。

## 功能

- PC 看板：事件队列、现场感知、AI 决策、处置链路、物联系统联动、人员通知、联动日志
- 手机看板：值班端事件确认、派发、呼叫与联动/通知查看
- 统计分析：风险结构、AI 置信度、闭环状态、处置阶段、联动执行率、通知触达率、区域热度

## 本地运行

```bash
npm install
npm run dev
```

## 验证

```bash
npm test
npm run build
```
